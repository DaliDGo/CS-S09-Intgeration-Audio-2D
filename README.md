# Intégration audio dans Unity2D
Projet démo pour l'intégration audio dans Unity2D

## Version du logiciel
Ce projet a été créé avec Unity2D version 2022.1.7f1.

## Assets utilisés
- 2D Platformer Microgame v2.1.0
